# weather-app
This is a simple weather app built with Python. It fetches real-time weather data from an API based on user input (like city name) and displays the current weather conditions.


Weather App

This is our weather application built with Python and PyQt5. It allows users to input a city name and retrieve the current temperature in Celsius along with a weather emoji.

Features
Input any city to get its weather.
Temperature displayed in Celsius.
Weather description with an emoji (e.g., clear, cloudy).
Prerequisites
Python 3.x
requests (pip install requests)
PyQt5 (pip install pyqt5)
Setup
Insert your OpenWeatherMap API key into the api_key variable in the code.
Running the App
Execute the script:
python weather_app.py
Enter a city name and click "Get Weather" to view the results.
Error Handling
Errors (e.g., invalid city) will display a clear message.
Ensure you have internet connectivity for API calls.
